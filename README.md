# Student User ID Generator

A simple Python program that generates a user ID for students using their:

- First Name
- Last Name
- Date of Birth
- Mobile Number

## User ID Format

User ID = First 2 letters of first name + Last 2 letters of last name + Last 2 digits of birth year + Last 4 digits of mobile number

### Example

```text
First Name: Rahul
Last Name: Sharma
DOB: 15-08-2005
Mobile: 9876543210

User ID: hama05210
```

## Requirements

- Python 3

## Run

```bash
python student_id_generator.py
```
